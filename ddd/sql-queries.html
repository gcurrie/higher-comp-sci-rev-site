SELECT and WHERE

The SELECT statement retrieves data from one or more tables. WHERE filters which rows are returned.

Basic SELECT
-- Return all columns from Pupil table
SELECT * FROM Pupil;

-- Return specific columns only
SELECT FirstName, LastName, Mark
FROM Pupil;

-- Filter with WHERE
SELECT FirstName, LastName, Mark
FROM Pupil
WHERE Mark >= 70;

-- Multiple conditions
SELECT FirstName, LastName
FROM Pupil
WHERE Mark >= 60 AND Mark < 70;

-- OR condition
SELECT * FROM Pupil
WHERE House = 'Red' OR House = 'Blue';

ORDER BY

ORDER BY sorts the result set. The default order is ascending (smallest first); add DESC for descending.

-- Alphabetically by last name (ascending)
SELECT FirstName, LastName FROM Pupil
ORDER BY LastName;

-- Highest mark first (descending)
SELECT FirstName, LastName, Mark FROM Pupil
ORDER BY Mark DESC;

-- Sort by multiple columns
SELECT FirstName, LastName, Mark FROM Pupil
ORDER BY LastName ASC, FirstName ASC;

Wildcards — LIKE

The LIKE operator is used with WHERE to search for a pattern rather than an exact match. Two wildcard characters are used:

WildcardMeaningExample
% Matches any sequence of zero or more characters 'Sm%' matches Smith, Smiley, Sm
_ Matches exactly one character '_one' matches Jones, Bones, Tones
LIKE Examples
-- All pupils whose surname starts with "Sm"
SELECT FirstName, LastName FROM Pupil
WHERE LastName LIKE 'Sm%';

-- All pupils whose email contains "@school"
SELECT FirstName, Email FROM Pupil
WHERE Email LIKE '%@school%';

-- All books published in the 1990s
SELECT Title, Year FROM Book
WHERE Year LIKE '199_';

-- All members whose name ends in "son"
SELECT * FROM Member
WHERE LastName LIKE '%son';

Computed Values and Aliases

SQL can calculate new values on the fly using arithmetic in the SELECT clause. An alias (using AS) gives the computed column a readable name in the output.

Computed Values with AS Alias
-- Calculate total price from unit price and quantity
SELECT ProductName,
       UnitPrice,
       Quantity,
       UnitPrice * Quantity AS TotalCost
FROM OrderLine;

-- Apply a 20% discount
SELECT ProductName,
       UnitPrice,
       UnitPrice * 0.8 AS DiscountedPrice
FROM Product;

-- Calculate percentage from marks
SELECT FirstName,
       Mark,
       ROUND((Mark / 100.0) * 100, 1) AS Percentage
FROM Pupil;
💡
The AS keyword gives the column a name in the results — it does not change the actual data in the table. You can use AS on any column, not just computed ones.

Aggregate Functions

Aggregate functions perform a calculation on a set of values and return a single result.

FunctionWhat it returnsExample
COUNT(*)Number of rowsHow many pupils are enrolled
COUNT(column)Number of non-NULL values in that columnHow many pupils have a mark recorded
SUM(column)Total of all valuesTotal sales for a product
AVG(column)Average (mean) of all valuesAverage mark in the class
MAX(column)Highest valueHighest mark achieved
MIN(column)Lowest valueLowest mark achieved
Aggregate Function Examples
-- Count all pupils
SELECT COUNT(*) AS NumberOfPupils FROM Pupil;

-- Average, max and min marks
SELECT AVG(Mark)  AS AverageMark,
       MAX(Mark)  AS HighestMark,
       MIN(Mark)  AS LowestMark
FROM Pupil;

-- Total value of all orders
SELECT SUM(UnitPrice * Quantity) AS TotalRevenue
FROM OrderLine;

GROUP BY

GROUP BY divides the rows into groups based on one or more columns, then applies aggregate functions to each group separately.

GROUP BY Examples
-- Count pupils in each House
SELECT House, COUNT(*) AS NumberOfPupils
FROM Pupil
GROUP BY House;

-- Average mark per year group
SELECT YearGroup,
       AVG(Mark) AS AverageMark,
       MAX(Mark) AS HighestMark
FROM Pupil
GROUP BY YearGroup
ORDER BY YearGroup;

-- Total sales per product
SELECT ProductName, SUM(Quantity) AS TotalSold
FROM Product
INNER JOIN OrderLine ON Product.ProductID = OrderLine.ProductID
GROUP BY ProductName
ORDER BY TotalSold DESC;
ℹ️
When using GROUP BY, every column in the SELECT clause must either be in the GROUP BY clause or be inside an aggregate function. You cannot mix grouped and non-grouped columns.
