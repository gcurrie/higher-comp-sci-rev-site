SQL — INSERT, UPDATE, DELETE

DDD Unit 2 Topic 6 of 7

As well as querying data with SELECT, SQL can modify the data in a database. The three commands for this are INSERT (add), UPDATE (change) and DELETE (remove).

⚠️
Be careful with UPDATE and DELETE: Always use a WHERE clause unless you intend to affect every record. Without WHERE, these commands affect the entire table.

INSERT INTO — Adding New Records

Adds a new row to a table. You list the field names and then the corresponding values:

INSERT INTO Pupil (PupilID, FirstName, LastName, YearGroup, Score, Passed)
VALUES (106, 'Callum', 'Ross', 5, 74, TRUE);

If you are inserting values for all fields in order, you can omit the field list:

INSERT INTO Pupil
VALUES (107, 'Niamh', 'Murray', 6, 83, TRUE);
💡
Text values must be enclosed in single quotes ('). Numbers and booleans do not need quotes.

UPDATE — Modifying Existing Records

Changes the value of one or more fields in existing records. Always use WHERE to specify which records to change:

-- Update the score for one pupil
UPDATE Pupil
SET Score = 88
WHERE PupilID = 102;

-- Mark all pupils as passed if their score is 50 or over
UPDATE Pupil
SET Passed = TRUE
WHERE Score >= 50;

-- Update multiple fields at once
UPDATE Pupil
SET Score = 70, Passed = TRUE
WHERE PupilID = 105;

UPDATE without WHERE — danger!

-- This updates EVERY record in the table!
UPDATE Pupil
SET YearGroup = 6;

DELETE — Removing Records

Permanently removes one or more records from a table. Always use WHERE:

-- Remove one specific pupil
DELETE FROM Pupil
WHERE PupilID = 105;

-- Remove all pupils who failed
DELETE FROM Pupil
WHERE Passed = FALSE;

-- Remove all Year 6 pupils
DELETE FROM Pupil
WHERE YearGroup = 6;

DELETE without WHERE — removes everything!

-- This deletes ALL records from the table (but keeps the table structure)
DELETE FROM Pupil;

Question 2 — 2 marks

Write a SQL statement to update the score of the pupil with PupilID 102 to 58 and set their Passed field to TRUE.

Question 3 — 2 marks

Write a SQL statement to delete all records from the Pupil table where the pupil's score is less than 40.

Question 4 — 2 marks
