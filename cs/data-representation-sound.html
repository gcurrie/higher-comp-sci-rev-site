Data Representation — Sound

Computer Systems Topic 3 of 6

Sound is naturally an analogue signal — a continuous wave that varies smoothly over time. To store sound digitally, it must be converted into a series of binary numbers through a process called Analogue-to-Digital Conversion (ADC).

How Sound is Digitised

The process of converting analogue sound to digital involves sampling:

  1. The amplitude (volume/height) of the sound wave is measured at regular intervals
  2. Each measurement is called a sample
  3. Each sample is stored as a binary number
  4. Playback reconstructs the wave from these samples

Key Terms

Sampling Rate
The number of samples taken per second. Measured in Hertz (Hz) or kilohertz (kHz). Higher sampling rate = more accurate reproduction of the original sound, but larger file size.
CD quality: 44,100 Hz (44.1 kHz)
Sample Depth (Bit Depth)
The number of bits used to store each sample. More bits = more possible amplitude values = higher quality, but larger file size.
CD quality: 16 bits per sample
Channels
Mono (1 channel) or stereo (2 channels). Stereo doubles the file size compared to mono.

Effect of Sampling Rate and Bit Depth

ChangeEffect on qualityEffect on file size
Increase sampling rateHigher frequency sounds reproduced more accuratelyLarger
Decrease sampling rateLoss of high-frequency detail; sound may seem muffledSmaller
Increase bit depthMore amplitude values; less distortion (quantisation noise)Larger
Decrease bit depthFewer amplitude values; distortion/noise more audibleSmaller

Calculating Sound File Size

File size (bits) = sampling rate × bit depth × duration (seconds) × number of channels

Example: 3-minute stereo audio at CD quality
Sampling rate = 44,100 Hz
Bit depth     = 16 bits
Duration      = 3 × 60 = 180 seconds
Channels      = 2 (stereo)

File size = 44,100 × 16 × 180 × 2
          = 254,016,000 bits
          = 254,016,000 ÷ 8        = 31,752,000 bytes
          = 31,752,000 ÷ 1024      = 31,007.8 KB
          = 31,007.8 ÷ 1024        ≈ 30.3 MB
💡
Units reminder: 1 byte = 8 bits | 1 KB = 1024 bytes | 1 MB = 1024 KB | 1 GB = 1024 MB. Always show your conversion steps in the exam.
