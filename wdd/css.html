CSS Basics

CSS (Cascading Style Sheets) controls the visual appearance of HTML elements. Rules are written as selector { property: value; }

CSS Rule Syntax
/* Selector targets element(s); rules inside braces */
h1 {
    color: #00442A;      /* font colour */
    font-size: 2rem;     /* font size */
    font-family: Arial, sans-serif;
}

/* Class selector */
.highlight {
    background-color: yellow;
    padding: 5px;
}

/* ID selector */
#main-nav {
    background-color: #002D1C;
}

Ways to Apply CSS

MethodHowBest for
External stylesheet<link rel="stylesheet" href="style.css">Production websites — reused across pages
Internal (embedded)<style> ... </style> in <head>Single-page demos
Inlinestyle="color:red" on the elementQuick overrides — avoid for large sites

Display Property

The display property controls how an element occupies space in the page flow.

ValueBehaviourExamples
block Takes up the full width available; starts on a new line; can have width/height/margin <div>, <p>, <h1>
inline Only as wide as its content; does NOT start a new line; cannot set width/height <span>, <a>, <strong>
inline-block Flows inline (no new line) BUT can have width, height and vertical margins Navigation items, image galleries
none Element is hidden completely — takes up no space Hiding dropdown menus, modal dialogs 
/* Make list items display side-by-side */
nav ul li {
    display: inline-block;
    margin-right: 10px;
}

/* Hide an element */
.hidden {
    display: none;
}

Float and Clear

float moves an element to the left or right, allowing text and other elements to wrap around it. It is commonly used to create multi-column layouts and navigation bars.

Float Example — Two-Column Layout
/* Left column — 70% width, floated left */
.main-content {
    float: left;
    width: 70%;
}

/* Right sidebar — 28% width, floated right */
.sidebar {
    float: right;
    width: 28%;
}

/* Footer — clear floats so it appears below both columns */
footer {
    clear: both;
}

The Clear Property

clear prevents an element from sitting beside a floated element. It forces the element below any floats on the specified side.

  • clear: left — clears left-floated elements
  • clear: right — clears right-floated elements
  • clear: both — clears floats on both sides (most common)
💡
Always add clear: both to the footer or any element that should appear below a floated layout — otherwise it will sit alongside the floated columns instead of below them.

Navigation Bars

Navigation bars are one of the most common CSS layout challenges. At Higher level, you need to be able to create both horizontal and vertical navigation bars.

Horizontal Navigation Bar (Using Float)

HTML
<nav>
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="about.html">About</a></li>
    <li><a href="contact.html">Contact</a></li>
  </ul>
</nav>
CSS
nav ul {
    list-style: none;   /* remove bullet points */
    margin: 0;
    padding: 0;
    overflow: hidden;   /* contain floated items */
    background-color: #00442A;
}

nav ul li {
    float: left;        /* place items side by side */
}

nav ul li a {
    display: block;     /* make link fill the li */
    padding: 14px 20px;
    color: white;
    text-decoration: none;
}

nav ul li a:hover {
    background-color: #C9A227;  /* highlight on hover */
    color: #002D1C;
}

Vertical Navigation Bar

/* For a vertical nav, remove float — block display stacks items */
nav ul li {
    display: block;    /* each item on its own line */
    border-bottom: 1px solid #ccc;
}

nav ul li a {
    display: block;
    padding: 10px 15px;
    color: #00442A;
    text-decoration: none;
}

nav ul li a:hover {
    background-color: #e8f5ee;
}

Hover Effects

The :hover pseudo-class applies styles when the user moves their mouse over an element. It is used extensively for navigation feedback, button effects and interactive elements.

/* Button hover effect */
.btn {
    background-color: #00442A;
    color: white;
    padding: 10px 20px;
    border: none;
    cursor: pointer;
}

.btn:hover {
    background-color: #C9A227;
    color: #002D1C;
}

/* Image hover — slight enlargement */
img:hover {
    transform: scale(1.05);
}

/* Link underline on hover */
a {
    text-decoration: none;
    color: #00442A;
}
a:hover {
    text-decoration: underline;
}

Box Model — Margins and Padding

Every HTML element is a rectangular "box". The CSS box model defines four areas around the content:

Margin (space outside border)
Border
Padding (space inside border)
Content
/* Shorthand: top, right, bottom, left */
.box {
    margin: 10px 20px 10px 20px;  /* all four sides */
    padding: 15px;                 /* same on all sides */
    border: 2px solid #00442A;
}

/* Individual sides */
.card {
    margin-top: 20px;
    margin-bottom: 20px;
    padding-left: 15px;
    padding-right: 15px;
}
PropertyEffect
marginSpace outside the border — pushes other elements away
paddingSpace inside the border — between border and content
borderLine around the element (width, style, colour)

Fonts and Colours

body {
    font-family: Arial, Helvetica, sans-serif; /* fallback chain */
    font-size: 16px;
    font-weight: normal;           /* or bold, 400, 700 */
    font-style: italic;            /* or normal */
    line-height: 1.6;              /* spacing between lines */
    color: #333333;                /* text colour */
}

h1 {
    font-size: 2rem;               /* relative to root font size */
    font-weight: bold;
    color: #00442A;
    text-align: center;
    text-transform: uppercase;     /* ALL CAPS */
    letter-spacing: 2px;
}

/* Background colours */
header {
    background-color: #002D1C;     /* hex colour */
    color: white;
}

.highlight {
    background-color: rgba(201, 162, 39, 0.2);  /* semi-transparent gold */
}
