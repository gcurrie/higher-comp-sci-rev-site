JavaScript in Web Pages

JavaScript makes web pages interactive. While HTML provides structure and CSS provides styling, JavaScript allows the page to respond to user actions — without requiring a page reload.

Including JavaScript

<!-- Method 1: External file (preferred) -->
<script src="script.js"></script>

<!-- Method 2: Inline in HTML (for small snippets) -->
<script>
  alert("Hello, world!");
</script>

<!-- Method 3: Event attribute (directly on element) -->
<button onclick="showMessage()">Click me</button>
💡
Place the <script> tag at the end of the body (before </body>) so that all HTML elements have loaded before JavaScript tries to access them.

document.getElementById

document.getElementById(id) is the most common way to select an HTML element in JavaScript. It returns the element whose id attribute matches the given string, allowing you to read or change its properties.

HTML
<p id="message">Original text.</p>
<img id="photo" src="default.jpg" alt="Photo" />
<div id="box" style="width:100px;height:100px;background:green;"></div>
JavaScript — Accessing and Modifying Elements
// Get an element by its id
let msg = document.getElementById("message");

// Read its text content
console.log(msg.innerHTML);   // "Original text."

// Change its text content
msg.innerHTML = "Text has been changed!";

// Change a CSS style property
let box = document.getElementById("box");
box.style.backgroundColor = "red";
box.style.width = "200px";

// Change an image source
let photo = document.getElementById("photo");
photo.src = "new-image.jpg";

Commonly Used Properties

PropertyWhat it gets/sets
element.innerHTMLThe HTML content inside the element (can include tags)
element.innerTextThe visible text content (no HTML tags)
element.style.propertyAn inline CSS style (e.g. .style.color, .style.display)
element.srcThe source URL of an image
element.classNameThe CSS class applied to the element

Mouse Events

JavaScript responds to user actions through event handlers. At Higher level, three mouse events are required:

EventWhen it fires
onclickWhen the user clicks on an element
onmouseoverWhen the mouse pointer moves onto an element
onmouseoutWhen the mouse pointer moves off an element

Events can be attached in two ways:

// Method 1: As an HTML attribute (inline event handler)
<button onclick="doSomething()">Click</button>

// Method 2: In JavaScript (addEventListener)
let btn = document.getElementById("myButton");
btn.addEventListener("click", doSomething);

function doSomething() {
    alert("Button clicked!");
}

onclick

The onclick event fires when the user clicks an element. It is used for buttons, links, images and any other clickable element.

onclick Examples
<!-- Toggle visibility of a section -->
<button onclick="toggleInfo()">Show / Hide Info</button>
<div id="info" style="display:none;">
  <p>This is the hidden information.</p>
</div>

<script>
function toggleInfo() {
    let info = document.getElementById("info");
    if (info.style.display === "none") {
        info.style.display = "block";
    } else {
        info.style.display = "none";
    }
}
</script>
onclick — Change Image Source
<img id="hero" src="day.jpg" alt="Scene" />
<button onclick="changeImage()">Switch Image</button>

<script>
function changeImage() {
    let img = document.getElementById("hero");
    if (img.src.endsWith("day.jpg")) {
        img.src = "night.jpg";
    } else {
        img.src = "day.jpg";
    }
}
</script>

onmouseover and onmouseout

onmouseover fires when the mouse enters an element; onmouseout fires when it leaves. Together they create rollover (hover) effects without CSS — useful when you need to respond to hover with JavaScript logic.

Image Rollover Effect
<img id="logo"
     src="logo-normal.png"
     alt="Logo"
     onmouseover="changeToHover()"
     onmouseout="changeToNormal()" />

<script>
function changeToHover() {
    document.getElementById("logo").src = "logo-hover.png";
}

function changeToNormal() {
    document.getElementById("logo").src = "logo-normal.png";
}
</script>
Tooltip / Info Box on Hover
<span id="hint-trigger"
      onmouseover="showHint()"
      onmouseout="hideHint()">Hover over me</span>
<div id="hint-box" style="display:none;background:yellow;padding:5px;">
  This is a hint!
</div>

<script>
function showHint() {
    document.getElementById("hint-box").style.display = "block";
}
function hideHint() {
    document.getElementById("hint-box").style.display = "none";
}
</script>

Changing Styles and Content

The combination of event handlers and document.getElementById allows JavaScript to change virtually any aspect of a page element dynamically.

Common Dynamic Changes
let el = document.getElementById("myElement");

// Change text
el.innerHTML = "New text content";

// Change background colour
el.style.backgroundColor = "#C9A227";

// Change font size
el.style.fontSize = "24px";

// Show / hide
el.style.display = "block";   // show
el.style.display = "none";    // hide

// Add/remove a CSS class
el.className = "highlighted";

// Change border
el.style.border = "3px solid red";
ℹ️
In JavaScript, CSS property names that contain a hyphen are written in camelCase: background-color → backgroundColor, font-size → fontSize, border-radius → borderRadius.
