Software Design & Development

Analyse, design, implement, test and evaluate software solutions using modular programming techniques, standard algorithms and structured development approaches.

ℹ️
SDD contributes 41–45 marks of the question paper and all 25 marks of Assignment Task 1. It is the most heavily assessed area — invest the most revision time here.

Higher builds directly on National 5 programming skills. The key additions at Higher are:

TopicHigher adds…
MethodologiesAgile methodology (compare with iterative).
AnalysisScope & Boundaries
DesignData flow in structure diagrams; wireframes for user-interface design
Implementation - Data structuresParallel 1D arrays; records; arrays of records (replaces simple 1D arrays)
Implementation - ConstructsFormal/actual parameter passing; local/global scope; pre-defined functions; file handling (CSV/txt)
Implementation - AlgorithmsApplied to Parallel arrays or arrays of records (linear search, find min/max, count occurrences)
TestingComprehensive final test plan; debugging using breakpoints and watchpoints
EvaluationUsability added as a criterion; robustness replaces readability
