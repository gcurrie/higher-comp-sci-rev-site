Data Types

A data type tells the program what kind of value a variable will hold and what operations can be performed on it. Choosing the correct data type is important for accuracy and efficiency.

Integer

Integers are used for counting, indexing and any situation where whole numbers are required. They cannot store decimal values — if a calculation produces a decimal result, it is truncated (not rounded).

# Python example
age = 16
score = 95
temperature = -3

Real (Floating-Point)

Real numbers (also called floats or doubles) are used when decimal precision is needed — for example, prices, measurements or scientific values. They use more storage than integers and can introduce small rounding errors due to binary representation.

# Python example
price = 4.99
height = 1.75
pi = 3.14159

Boolean

Booleans store a logical true/false value. They are commonly used in conditions (if statements) and flags. Named after mathematician George Boole.

# Python example
logged_in = True
game_over = False

if logged_in:
    print("Welcome back!")

Character & String

A character is a single symbol. A string is a sequence of characters treated as a single unit. Strings support operations like concatenation (joining), length, and slicing.

# Python example
initial = 'J'          # character
name = "John Smith"    # string
greeting = "Hello, " + name   # concatenation
print(len(name))       # length = 10

Key Terms

Variable
A named location in memory used to store a value that can change during program execution.
Literal
A fixed value written directly in code, e.g. 42 or "hello".
Concatenation
Joining two or more strings together, e.g. "Hello" + " World".
Casting
Converting a value from one data type to another, e.g. int("5") converts the string "5" to the integer 5.
