Standard Algorithms

At Higher level, three standard algorithms are required: linear search, find minimum and maximum, and count occurrences. These can be applied to 1D arrays or arrays of records.

Which Algorithms Are Required at Higher?

⚠️

Higher requires only three standard algorithms:

  • Linear search
  • Find minimum and maximum
  • Count occurrences

Input validation and running total are National 5 algorithms — they will not be asked in a Higher exam context as a "standard algorithm" question.

Linear Search

Linear search is used with one or more arrays of items. We traverse (loop through) the array, looking for a particular value. If one of the items in the array matches the search value, we do something.

There are two common variations: one records whether something is found, while the other records where it was found (its position in the list).

Recording Whether an Item is Found

Python — Linear Search (found flag)
names = ["Dopey", "Grumpy", "Doc", "Bashful", "Sneezy", "Sleepy", "Happy"]
target = input("Enter a name to look for")
found = False

for counter in range(0, 7):
  if names[counter] == target:
    found = True

print("Item was found:", found)

Recording the Position of an Item

Python — Linear Search (position)
names = ["Dopey", "Grumpy", "Doc", "Bashful", "Sneezy", "Sleepy", "Happy"]
target = input("Enter a name to look for")
position = -1

for counter in range(0, 7):
  if names[counter] == target:
    position = counter

print("Item was found at position:", position)
💡
The position is initialised to -1 because it isn't possible to be at position -1 in the array. That way, we know that anything other than -1 must mean the item was found.

Searching Parallel Arrays

Linear search is often used to find an item in one array, then read the corresponding data from a second, parallel array.

Python — Linear Search Across Parallel Arrays
pupils = ["Fry", "Bender", "Leela", "Hermes", "Zoidberg"]
marks = [10, 2, 1, 8, 9]
target_name = input("Please enter a name: ")
position = -1

for counter in range(len(pupils)):
  if pupils[counter] == target_name:
    position = counter

their_mark = marks[position]

if position != -1:
  print(target_name + " got the mark " + str(their_mark))
else:
  print("Target not found")

Find Maximum

The find maximum (find-max) and find minimum (find-min) algorithms are very similar, only differing in whether they use the > or < sign.

To find the maximum, we assume the first item is the largest, then loop through the rest of the array. Whenever we find a larger value, we update our maximum.

Pseudocode

  1. Declare a maximum variable, set it to the first item in the array
  2. Loop for each element in the array
  3. If number(counter) > maximum
  4. Set maximum to number(counter)
  5. End if
  6. End for loop
Python — Find Maximum
numbers = [33, 44, 55, 12, 19, 72, 60, 54, 13, 98, 2, 75]
maximum = numbers[0]
for index in range(1, len(numbers)):
  if numbers[index] > maximum:
    maximum = numbers[index]
print("The highest number is", maximum)

Find Minimum

Finding the minimum works in exactly the same way as finding the maximum — the only change is that we use the < sign instead of >. We assume the first item is the smallest, then update our minimum whenever we find a smaller value.

Pseudocode

  1. Declare a minimum variable, set it to the first item in the array
  2. Loop for each element in the array
  3. If number(counter) < minimum
  4. Set minimum to number(counter)
  5. End if
  6. End for loop
Python — Find Minimum
numbers = [33, 44, 55, 12, 19, 72, 60, 54, 13, 98, 2, 75]
minimum = numbers[0]
for index in range(1, len(numbers)):
  if numbers[index] < minimum:
    minimum = numbers[index]
print("The lowest number is", minimum)

Using Find Minimum with Parallel Arrays

As with linear search, find minimum is often used with parallel arrays — here we track the name of the youngest person as well as their age.

Python — Youngest Person (Parallel Arrays)
names = ["Sarah", "Martin", "Gill", "Andrew", "Gerald", "James"]
ages = [15, 32, 40, 12, 72, 16]
minimum = ages[0]
youngest_name = names[0]
for count in range(0, 6):
  if ages[count] < minimum:
    minimum = ages[count]
    youngest_name = names[count]
print("The youngest person is", youngest_name)

Finding Both at Once

Because the two algorithms are so similar, you can find the minimum and maximum together in a single loop:

Python — Find Minimum and Maximum Together
numbers = [33, 44, 55, 12, 19, 72, 60, 54, 13, 98, 2, 75]
minimum = numbers[0]
maximum = numbers[0]
for index in range(1, 12):
  if numbers[index] < minimum:
    minimum = numbers[index]
  if numbers[index] > maximum:
    maximum = numbers[index]
print("The lowest number is", minimum)
print("and the highest is", maximum)

Count Occurrences

Count occurrences is very similar to linear search. The user enters a target/search term. The algorithm then searches the array. If the target is found, instead of recording position, it adds +1 to a total/counter.

The algorithm can search for exact matches, or apply conditional logic — values above a threshold, even numbers, uppercase letters, and so on.

⚠️
Do not confuse counting occurrences with keeping a running total: one counts how many of something there are, the other adds them all together.

Counting Exact Matches

Python — Count Matching Car Makes
cars = ["Ford", "Ford", "Toyota", "Volkswagen", "Kia", "Nissan", "Honda"]
make = input("Please enter a make: ")
count = 0
for index in range(len(cars)):
  if cars[index] == make:
    count = count + 1
print("There were", count, "matching cars")

Counting with a Condition (Range Check)

Python — Count Values of 50% or More
percentages = [99.7, 100.0, 52.6, 13.9, 15.2, 88.1, 64.7, 22.5, 71.8]
target = 50.0
count = 0
for index in range(len(percentages)):
  if percentages[index] >= target:
    count = count + 1
print("There were", count, "values of 50% or more")

Counting into Two Categories (Odd / Even)

Python — Count Odd and Even Numbers
numbers = [99, 44, 55, 12, 19, 72, 60, 54, 13, 18, 2, 75]
odd_count = 0
even_count = 0
for index in range(len(numbers)):
  if (numbers[index] % 2) == 0:
    even_count = even_count + 1
  else:
    odd_count = odd_count + 1
print("There were", even_count, "even numbers")
print("and", odd_count, "odd numbers in the list")

Applying Algorithms to Arrays of Records

All three algorithms work with arrays of records — you simply specify which field to compare.

Find Maximum in an Array of Records
RECORD Pupil IS
    name : STRING
    mark : INTEGER
END RECORD

DECLARE class AS ARRAY OF Pupil INITIALLY []
# (assume class is populated with pupil data)

DECLARE maxIndex AS INTEGER INITIALLY 0

FOR i FROM 1 TO LENGTH(class) - 1 DO
    IF class[i].mark > class[maxIndex].mark THEN
        SET maxIndex TO i
    END IF
END FOR

SEND "Top pupil: " & class[maxIndex].name &
     " with " & class[maxIndex].mark TO DISPLAY
Linear Search on a Record Field
DECLARE target AS STRING INITIALLY "Bob"
DECLARE found AS BOOLEAN INITIALLY FALSE
DECLARE foundAt AS INTEGER INITIALLY -1

FOR i FROM 0 TO LENGTH(class) - 1 DO
    IF class[i].name = target THEN
        SET found TO TRUE
        SET foundAt TO i
    END IF
END FOR
