📋About This Course

PLEASE NOTE, ALL MATERIAL HERE IS CURRENTLY PLACEHOLDER AND IS BEING UPDATED.

Higher Computing Science is assessed through two components: a question paper and a coursework assignment. Together these are worth 120 marks. The course covers four areas of study, with candidates choosing either Database Design & Development or Web Design & Development as their optional area.

Use this site to work through each unit topic by topic. Each section includes explanations, diagrams, worked examples and practice questions aligned to the SQA Higher course specification (Version 3.0).

📚Course Areas

💻
Mandatory

Software Design & Development

Design, implement, test and evaluate software solutions. Covers modular programming, data structures, pre-defined functions, file handling, standard algorithms and development methodologies.

  • Development Methodologies
  • Analysis
  • Design (structure diagrams, pseudocode, wireframes)
  • Data Types & Structures (1D arrays, records, arrays of records)
  • Computational Constructs & Pre-defined Functions
  • Standard Algorithms
  • Testing & Debugging
  • Evaluation

Mandatory

Computer Systems

Data representation, computer structure, the fetch-execute cycle, performance factors, environmental impact of intelligent systems, and security legislation and encryption.

  • Two's Complement & Floating-Point
  • Unicode, Bitmap & Vector Graphics
  • Fetch-Execute Cycle & Performance Factors
  • Environmental Impact of Intelligent Systems
  • Computer Misuse Act 1990
  • DOS Attacks & Tracking Cookies
  • Encryption (Public/Private Keys, Digital Certificates)

🗄️
Optional

Database Design & Development

Design relational databases with three or more linked tables. Covers ER diagrams, entity-occurrence diagrams, compound keys, data dictionaries and SQL.

  • Analysis & End-user Requirements
  • ER Diagrams (3+ entities) & Cardinality
  • Entity-Occurrence Diagrams & Compound Keys
  • Data Dictionary & Validation
  • SQL — SELECT with Aggregates, Aliases, Wildcards
  • SQL — INNER JOIN (3+ tables)
  • SQL — INSERT, UPDATE, DELETE

Optional

Web Design & Development

Design and build multi-level websites using HTML, CSS and JavaScript. Covers wireframing, prototyping, CSS layout and navigation, HTML forms, JS mouse events and usability testing.

  • Website Structure & Wireframing
  • Prototyping (Low Fidelity)
  • CSS — Display, Float, Navigation Bars
  • HTML — Semantic Elements & Forms
  • JavaScript — Mouse Events
  • Usability Testing (Personas, Test Cases)
  • Compatibility Testing
